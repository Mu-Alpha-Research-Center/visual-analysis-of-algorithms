# TypeScript Algorithms

## Getting Started

-   Install [nvm](https://github.com/nvm-sh/nvm)
-   `nvm install && nvm use && corepack enable`
-   `yarn && yarn test`

## Commands

| Command                             | Description                            |
| ----------------------------------- | -------------------------------------- |
| `yarn test`                         | Watch tests                            |
| `yarn generate:leetcode-problem -h` | Generate new LeetCode problem template |
